# 🚀 Portfolio V2 — Mishti Agarwal

A futuristic, cinematic, and performance-optimized developer portfolio built with Next.js, React, TypeScript, GSAP, Lenis, and React Three Fiber.

Designed to showcase projects, open-source contributions, technical skills, and learning journey through an immersive deep-space inspired user experience.

---

## 🌌 Overview

Portfolio V2 is a modern single-page portfolio focused on:

* Interactive 3D experiences
* Smooth scrolling interactions
* Glassmorphic user interfaces
* Custom cursor animations
* Bento-grid layouts
* High-performance rendering
* Responsive design
* Accessibility and SEO optimization

The visual language combines futuristic operating-system aesthetics, cinematic motion design, and deep-space inspired interfaces.

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/mishtiagrawal02-cloud/portfolio.git
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build production version:

```bash
npm run build
```

Run production server:

```bash
npm start
```
